Image Processing

SkyBits is actively using Deep Learning based techniques Convolution Neural Network (CNN) for automatic feature extraction and fully connected neural nets for subsequent classification. Popular Python based tools Keras on top of Theano & TensorFlow, OpenCV libraries have been used extensively to achieve high accuracy in detection & prediction.

Natural Language Processing

Along with the growth of unstructured text data, NLP techniques have grown from statistical based techniques of TF, IDF, Bag of Words to more sophisticated techniques like text corpus vectorization, deep learning based named entity recognition. SkyBits is working extensively in this area for last two years and using advanced Python NLP platform Spacy, Keras on top of Theano/TensorFlow.

Statistical Analysis

SkyBits works with government departments in analysis and statistical modelling from large structured dataset.Standard big data tools like Apache Spark is used for initial descriptive data analytics for raw data and processed data.

Mobile Applications

SkyBits is working on end-to-end solutions having mobile devices based applications and cloud based server applications for last four years. These solutions are developed as products for its customers primarily in e-Learning domain.
